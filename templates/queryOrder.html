行云订单API v3.1

查看订单状态

简要描述:

  • 接口地址：*.*.*.*/api/service/business
  • 支持格式：JSON
  • 请求方式：HTTP POST
请求参数说明
名称 类型 必填 说明
opcode String 是 操作码，固定值: get_order_info
merchant_id String 是 商户id
order_no String 是 商品订单编号(分单)
sign_type String 是 加密方式:MD5(大写)
sign String 是 MD5加密串
返回参数说明
名称 类型 说明
ret_code String 业务结果返回
ret_msg String 业务结果描述返回
sign String MD5加密串
sign_type String 加密方式:MD5(大写)
order_status String 订单状态
express_no String 快递单号(发货返回)
express_cpy String 快递公司(发货返回)
express_cpy_code String 快递公司编码(发货返回)
freight_amount String 运费(分)
price String 单价(分)
buy_num String 购买数量
order_price String 订单总金额(包括运费)
