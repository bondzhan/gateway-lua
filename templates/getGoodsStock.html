获取商品库存

示例:

  • 接口地址：***.**.**.**/api/service/business
  • 支持格式：JSON
  • 请求方式：HTTP POST
请求参数说明
名称 类型 必填 说明
opcode String 是 操作码，固定值:get_goods_stock
merchant_id String 是 商户id
sku_list String 是 Skuid商品ID,用,号隔开
sign_type String 是 加密方式:MD5(大写)
sign String 是 MD5加密串

JSON请求示例


{
        "sku_list":"60002,60003",
        "sign":"c6ffd8c3ffbf7e72c3d4c9a506146be7",
        "merchant_id":"75041628",
        "opcode":"get_goods_stock",
        "sign_type":"MD5"
}
返回参数说明
名称 类型 说明
ret_code String 业务结果返回
ret_msg String 业务结果描述返回
sign String MD5加密串
sign_type String 加密方式:MD5(大写)
stock_info sku_id String 商品编码
sku_stock_num_show String 库存数量

JSON返回示例


{
    "sign":"9fe764d29f5bd2421a9757dce545f2c1",
    "stock_info":"[
                    {"sku_id":"60002","sku_stock_num_show":564},
                    {"sku_id":"60003","sku_stock_num_show":426}
    ]",
    "ret_msg":"get_goods_stock查询成功！",
    "ret_code":"100002",
    "sign_type":"MD5"
}
