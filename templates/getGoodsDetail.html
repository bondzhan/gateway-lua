获取产品详情

简要描述:

  • 接口地址：***.**.**.**/api/service/business
  • 支持格式：JSON
  • 请求方式：HTTP POST
请求参数说明
名称 类型 必填 说明
opcode String 是 操作码，固定值:goods_detail
merchant_id String 是 商户id
sku_id String 是 Skuid商品ID
sign_type String 是 加密方式:MD5(大写)
sign String 是 MD5加密串
返回参数说明
名称 类型 说明
ret_code String 业务结果返回
ret_msg String 业务结果描述返回
sign String MD5加密串
sign_type String 加密方式:MD5(大写)
sku_id String 商品编码
sku_name String 商品名称
sku_suffix String 商品名称后缀
sku_prefix String 商品名称前缀
sku_stock_num_show String 商品库存
sku_detail String 商品详情
sku_subTitle String 商品副标题
freight_id String 运费模板ID
sku_weight String 商品重量
sku_weight_unit String 商品重量单位
sku_pic String 商品图片列表以,号隔开
sku_international_no String 商品国际编码
min_no String 最小起发数
max_no String 最大起发数
quality_start_date String 保质期开始时间
quality_end_date String 保质期结束时间
model_type String 模板类型1全国包邮，2部分地区包邮，3不包邮
ship_city String 发货地
trade_id String 贸易类型: 1保税商品，2直邮商品，3完税商品，4售后商品
brand_name String 品牌名
brand_logo String 品牌logo图
country_name String 原产地
country_logo String 国旗logo
country_map String 国家地图
price_range_list start_num String 起发头
end_num String 起发尾
range_price String 起发价
