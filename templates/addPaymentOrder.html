行云订单API v3.1

{(./nav.html)}

增加待支付订单

简要描述:

  • 接口地址：***.**.**.**/api/service/business
  • 支持格式：JSON
  • 请求方式：HTTP POST
请求参数说明
名称 类型 必填 说明
opcode String 是 操作码，固定值: add_unpaid_order
merchant_id String 是 商户id
merchant_order_no String 是 订单标识Id(一般是对接方的订单号)
accept_name String 是 收货人姓名
card_id String 是 身份证号码
post_code String 是 邮政编码
telphone String 否 联系电话
mobile String 是 手机
province String 是 所属区域：省
city String 是 所属区域：市
area String 是 所属区域：区
address String 是 收货地址
card_url_front String 否 身份证正面URL部分直邮必填
card_url_back String 否 身份证背面URL部分直邮必填
message String 否 订单留言
items sku_id String 是 商品编码(SKU_ID)
quantity int 是 订购数量
sign_type String 是 加密方式:MD5(大写)
sign String 是 MD5加密串
返回参数说明
名称 类型 说明
ret_code String 业务结果返回
ret_msg String 业务结果描述返回
sign String MD5加密串
sign_type String 加密方式:MD5(大写)
merchant_order_no String 外部订单标识(您传入的订单号)
goods_orders order_no String 商品订单(拆分的订单)
sku_id String sku编码
freight_amount String 运费（分）
goods_price String 单价（分）
buy_num String 购买数量
